Multiselect

Multi selection on the select tag without holding the <ctrl> button; the plugin doesn't add any additional HTML.

  • How to build it: The source file is src-min/multiselect-min.js: this is the only file you need to include in your pages. In the demo folder there is a demo html page; the demo needs the filesystem tree exactly as it is on the repository (folders "src-min", "demo" and "dependencies"). In the dependencies folder there is the jQuery framework, version 1.7.2: this is the version i used in my tests.
